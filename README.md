# Gestão de Funcionários em Python

Sistema simples de gestão de funcionários desenvolvido em Python, utilizando listas e dicionários para armazenamento dos dados.

## Funcionalidades

- Adicionar funcionário (nome, cargo e salário)
- Excluir funcionário pelo índice
- Listar todos os funcionários cadastrados
- Encerrar o programa

## Estrutura dos dados

Cada funcionário é representado por um dicionário no formato:
- `nome` (string)
- `cargo` (string)
- `salario` (float)

Os funcionários são armazenados em uma lista chamada `funcionarios`.

## Execução

1. Tenha o Python 3 instalado.
2. Execute o arquivo no terminal:
3. Escolha uma opção pelo menu numérico.

## Objetivo do projeto

Praticar:
- Estruturas de dados (listas e dicionários)
- Laços de repetição
- Entrada e saída de dados
- Lógica básica de CRUD em Python

## Observações

Projeto educacional, sem persistência de dados em arquivo ou banco de dados.
Os dados são perdidos ao encerrar o programa.
